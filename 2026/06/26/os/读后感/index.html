# 本科生关于读博 AB 面的读后感

# 1. 引子

《我们为什么要读博士？》《昨夜无眠》《我为什么鼓励你读博士？》(以下简称为 ref1、ref3、ref2)，这三篇文章分别从博士生、博士生导师以及一位计算机教授的角度，阐述了对读博的态度。作为一名刚上大学未满两年的本科生，虽然第三篇与我的专业关联最大，对我来说，印象最深的是前两篇。

# 2. 科学家之梦与现实之差

为什么读博？5 年前我一定会说为了成为一名科学家。但现在的我的 answer 大概率是为了能够拿顶级企业的 offer，然后美美高薪。与 ref1 作者不同的是，我本身不擅长基础学科，所以选择了网络空间安全专业，虽然之前我从未过多接触过计算机。牛顿说 “自己好像一个在海边玩耍的孩子，只是偶尔拾到一颗光滑美丽的石子，于是也向做一个一辈子在海边玩耍的孩子”，其实谁不是 “捡石子的小孩”，不过是学会了拿得起又放得下。兴趣是最好的老师？我觉得更多是幸存者偏差罢了，古往今来能够坚持初衷，在兴趣领域出彩之人所占几何？诚如 ref1 作者所言，即使怀着兴趣，读博后也可能会发现 “捡石子” 也有 KPI。到头来却怀疑自己开始的选择。本人家庭并不富裕，也不像达芬奇一样，与 “法国国王” 是至交。父母老去，我不希望出现 “久病床前无孝子”、“子欲养而亲不待”、想帮助他人却无能为力这种情况。尽管片面刻薄，但就是如此，先站稳了，才能谈顶天立地的事儿。

# 3. 读博的困境与抉择

ref3 从博士生导师的角度讲述，所言皆是对失去得意门生的悲伤，但在评论区中我最认同的一句话：人各有志。人所追求的目标一定是在变化的，早上起床想有个好天气，中午饿了想吃顿大餐，晚上困了想睡个好觉。站在那个博士的角度，尽管科研能力惊才绝艳，长期高强度工作，难免心生其他想法。想来读博就像进高压锅，不是变强就是变熟，就算是凤凰，也大有可能变烧鸡。有金刚钻不是读博的唯一标准，志之所趋，心之所向，不同时期不同境遇，都有不同的志向。

# 4. 计算机读博的有利形势与自我思考

实话说 ref2 给咱计算机人很大鼓励。现在几乎各行各业都有计算机技术参与，在应用这一方面，计算机专业没得说。现在说 AI 带来计算机寒冬，其实是计算机划时代的发展，绝对不是计算机的寒冬，而是跟不上发展的人的寒冬。我很喜欢作者说计算机读博是一个进可攻退可守的选择，进可科研成才，退可企业高薪，计算机博士就像漫威的奇异博士，可以预见多个未来结果。除了技术门槛，计算机专业读博更多需要非技术能力提升：交流表达、克服苦难，实际中，大部分学计算机的人都不善于交流表达。

身为计算机类专业的本科生，虽没有化学、数学那些基础学科的科研深造压力，而且坐拥进可攻退可守的有利选择，但也是焦虑的。对比作者和作者所提人物，我见其人如蜉蝣见青天，但就如上图一样，在未来遥远，能力不足的情况下焦虑是没有意义的，做好眼前事，走好脚下路才是最正确的。
