# 【动态规划】最长回文子串

# 题目

给你一个字符串 s，找到 s 中最长的 回文 子串。

示例 1：
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。

示例 2：
输入：s = "cbbd"
输出："bb"

提示：

1 <= s.length <= 1000
s 仅由数字和英文字母组成

# dp [i][j] 表格示例：
假设 s 为 ababaaba

0truefalsetruefalsetruefalsefalsefasle
1truefalsetruefaslefalsefalsefalse
2turefalsetruefalsefalsetrue
3turefalsefalsetruefalse
4turetruefalsefalse
5turefalsetrue
6turefalse
7ture

  • 在这个表格中可以看出，我们只填写对角线上半矩阵，因为子串开始位置一定小于终止位置。

  • 对于动态规划最重要的信息是构建状态转移方程，这里我们发现如果 s [i]!=s [j]，则说明 s [i..j] 一定不是回文子串，所以直接 dp [i][j]=false 即可，但当 s [i]==s [j] 时，dp [i][j] 的值取决于其左下角 dp [i+1][j-1] 的值，也就是 s [i..j] 要是回文串的前提时 s [i+1..j-1] 是回文子串，这样就构造出了状态转移方程。

  • 主对角线上的元素一定是 true，因为一个字符本身就是回文串，但是这个条件没啥用。

  • 动态规划的边界是回文串长度 <=2 时，如果 s [i] s [j]，则说明要么 ij，也就是 s [i] 和 s [j] 是同一个字符，要么 s [i] 和 s [j] 相邻且相同，构成 aa 这样的回文串，由此我们遍历 i、j 填表的顺序秩序满足 dp [i][j] 可以访问 dp [i+1][j-1]，也就是可以访问元素左下角相邻元素即可，由此完成完整动态规划。

  • 这里可以按两种方式遍历这个表：

    1. 行从下往上，列从左往右
    2. 列从右往左，行从上往下
