# 散列函数

# 散列函数的概念

散列函数又称为杂凑函数，也叫哈希函数（Hash function）。以下均称为 Hash 函数，用 H 表示。
Hash 函数算法公开，没有密钥。
作用是可以将任意长度的输入 （M） 映射到一个固定长度的输出 （MD） , 即： H(M)=MD 。
Hash 函数 - 碰撞问题：Hash 是多对一的映射，对于同一 MD，可能存在多个 M。也就是 x≠y满足H(x)=H(y) 。

# 散列函数的基本性质

  1. 输入长度可变，输出长度固定。
  2. 抗原像攻击 (单向性)：对任何给定的散列码 md，寻找 x 使得 H (x)＝md 在计算上是不可行的。
  3. 抗第二原像性 (抗弱碰撞性)：对任何给定的分组 x，寻找不等于 x 的 y，使得 H (x)=H (y) 在计算上是不可行的。
  4. 抗碰撞性 (抗强碰撞性)：寻找任何 (x,y) 对 (x≠y)，使得 H (x)＝H (y) 在计算上是不可行的。

# 散列函数的主要特性

  1. 散列函数的输出不以可辩别的房四海依赖于输入。
  2. 强混淆特性：输入串中的单个 bit 变化会导致输出串的大约一半 bit 发生变化。
  3. 散列函数具有确定性：散列值不同 => 输入串不同。
  4. 多对一映射：可能出现哈希碰撞。

# 散列函数的主要应用

  1. 保证数据完整性
  2. 用于数字签名
  3. 单向数据加密：口令表加密，存储口令的哈希值。
