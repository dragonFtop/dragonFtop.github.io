# 计算机网络与因特网作业答案及解析

# 1. 单选题

题目：
In the following options, which does not define in protocol?
A. the format of messages exchanged between two or more communicating entities
B. the order of messages exchanged between two or more communicating entities
C. the actions taken on the transmission of a message or other event
D. the transmission signals are digital signals or analog signals

答案：D
解析：
网络协议的三要素是语法（消息格式）、语义（动作响应）、时序（消息顺序）。

  • A 对应语法，B 对应时序，C 对应语义，均属于协议定义的内容；
  • D 描述的是传输信号的类型，属于物理层的传输介质特性，不属于协议本身的定义。

# 2. 单选题

题目：
In the following options, which is defined in protocol?
A. the actions taken on the transmission and/or receipt of a message or other event
B. the objects exchanged between communicating entities
C. the content in the exchanged messages
D. the location of the hosts

答案：A
解析：

  • A 属于协议的语义部分，定义了通信实体在发送 / 接收消息时需要执行的动作，是协议的核心要素；
  • B（交换的对象）、C（消息内容）、D（主机位置）均由应用层业务决定，不属于协议的定义范畴。

# 3. 单选题

题目：
An application can rely on the connection to deliver all of its data without error and in the proper order. The sentence describes ( ).
A. flow control
B. congestion-control
C. reliable data transfer
D. connection-oriented service

答案：C
解析：

  • reliable data transfer（可靠数据传输）：核心定义就是保证数据无差错、按顺序、完整交付，与题干描述完全匹配；
  • flow control（流量控制）：防止发送方速率超过接收方处理能力；
  • congestion-control（拥塞控制）：防止网络拥塞导致丢包；
  • connection-oriented service（面向连接服务）：是实现可靠传输的一种方式，但不等同于可靠传输本身。

# 4. 单选题

题目：
The Internet's connection-oriented service has a name, it is ( ).
A. TCP
B. UDP
C. TCP/IP
D. IP

答案：A
解析：

  • TCP（传输控制协议）：是互联网中典型的面向连接服务，提供可靠、有序、无差错的数据传输；
  • UDP（用户数据报协议）：无连接服务；
  • TCP/IP：是整个协议簇的统称，不是单一服务；
  • IP（网际协议）：无连接的网络层协议。

# 5. 单选题

题目：
The Internet's connectionless service is called ( ).
A. TCP
B. UDP
C. TCP/IP
D. IP

答案：B
解析：

  • UDP（用户数据报协议）：是互联网中典型的无连接服务，不建立连接、不保证可靠交付，适用于实时性要求高的场景；
  • TCP：面向连接；
  • TCP/IP：协议簇统称；
  • IP：网络层无连接协议，但题干问的是传输层的无连接服务，UDP 是更准确的答案。

# 6. 单选题

题目：
Which of the following nodes belongs to the network core?
A. a Web Server
B. a Host with Win2003 Server
C. a Router with NAT service
D. a Supernode on Skype Network

答案：C
解析：
网络核心（Network Core）指互联网中负责路由转发、数据交换的核心设备，不直接面向用户；边缘部分（Network Edge）是用户终端、服务器等。

  • A、B 属于边缘的主机 / 服务器；
  • C（路由器）是网络核心的核心设备，负责跨网络数据转发；
  • D（Skype 超级节点）属于 P2P 网络的边缘节点，不属于互联网核心。

# 7. 单选题

题目：
In the Internet, the equivalent concept to end systems is ( ).
A. hosts
B. servers
C. clients
D. routers

答案：A
解析：

  • end systems（端系统）：在互联网中与 hosts（主机） 是完全等价的概念，包含所有用户终端、服务器等；
  • B（服务器）、C（客户端）只是主机的子集；
  • D（路由器）属于网络核心，不是端系统。

# 8. 单选题

题目：
In the Internet, end systems are connected together by ( ).
A. copper wire
B. coaxial cable
C. communication links
D. fiber optics

答案：C
解析：

  • communication links（通信链路） 是统称，包含铜线、同轴电缆、光纤、无线链路等所有传输介质；
  • A、B、D 只是具体的链路类型，不能涵盖所有连接方式，因此 C 是最准确的答案。

# 9. 单选题

题目：
End systems access to the Internet through its ( ).
A. modems
B. protocols
C. ISP
D. sockets

答案：C
解析：

  • ISP（互联网服务提供商） 是端系统接入互联网的唯一入口，用户通过 ISP 的网络接入互联网；
  • A（调制解调器）是接入设备，不是接入主体；
  • B（协议）是通信规则，不是接入方式；
  • D（套接字）是应用层编程接口，与接入无关。

# 10. 单选题

题目：
In the following options, which belongs to the network core? ( )
A. end systems
B. routers
C. clients
D. servers

答案：B
解析：
网络核心由路由器、交换机等转发设备组成，负责数据的路由和交换；

  • A（端系统）、C（客户端）、D（服务器）均属于网络边缘部分，不属于核心。

# 11. 单选题

原题：
End systems, packet switches, and other pieces of the Internet run () that control the sending and receiving of information within the Internet.
A. programs
B. processes
C. applications
D. protocols

答案：D
解析：网络中所有设备的通信规则由协议（protocols） 控制，这是网络通信的核心规范。A（程序）、B（进程）、C（应用）是协议的运行载体，并非控制通信的核心规则。

# 12. 单选题

原题：
The internet allows () running on its end systems to exchange data with each other.
A. clients applications
B. server applications
C. P2P applications
D. distributed applications

答案：D
解析：互联网的核心设计目标是支持分布式应用（distributed applications） 在端系统间交换数据。A、B、C 仅为分布式应用的特定类型，无法全面涵盖所有场景。

# 13. 单选题

原题：
The protocols of various layers are called ().
A. the protocol stack
B. TCP/IP
C. ISP
D. network protocol

答案：A
解析：网络各层协议的集合被称为协议栈（the protocol stack）。B 是具体协议簇，C 是互联网服务提供商，D 是网络协议统称，均不符合 “各层协议集合” 的定义。

# 14. 单选题

原题：
The lower layers of the OSI model are, in correct order
A. physical, system, network, logical
B. physical, logical, network, system
C. physical, transport, network, data link
D. physical, data link, network, transport

答案：D
解析：OSI 模型底层从下到上的正确顺序为：物理层→数据链路层→网络层→传输层。A、B 概念错误，C 顺序颠倒，均不符合 OSI 模型层级定义。

# 15. 单选题

原题：
The Internet Protocol (IP) generally corresponds to which OSI layer?
A. Network (layer three)
B. Transport (layer four)
C. Data link (layer two)
D. Session (layer five)

答案：A
解析：IP 协议负责网络层的路由寻址，对应 OSI 的网络层（第三层）。传输层对应 TCP/UDP，数据链路层对应以太网等，会话层与 IP 无关。

# 16. 单选题

原题：
What layer of the OSI model is designed to perform error detection functions?
A. Physical
B. Data link
C. Network
D. transport

答案：B
解析：数据链路层的核心功能之一是错误检测（如通过 CRC 校验）。物理层侧重信号传输，网络层负责路由，传输层侧重可靠传输，均不是错误检测的核心层。

# 17. 单选题

原题：
Which of the following protocol layers is not explicitly part of the Internet Protocol Stack? ______
A. application layer
B. session layer
C. data link layer
D. transport layer

答案：B
解析：互联网协议栈（TCP/IP 模型）包含应用层、传输层、网络层、数据链路层，不包含会话层。会话层是 OSI 模型的冗余层，未被纳入 TCP/IP 协议栈。

# 18. 单选题

原题：
The 5-PDU is called __
A. message
B. segment
C. datagram
D. frame

答案：A
解析：OSI 模型中，第五层（会话层）的协议数据单元（PDU）称为消息（message）。分段（segment）是传输层 PDU，数据报（datagram）是网络层 PDU，帧（frame）是数据链路层 PDU。

# 19. 单选题

原题：
The Internet’s network layer is responsible for moving network-layer packets known as () from one host to another.
A. frame
B. datagram
C. segment
D. message

答案：B
解析：网络层负责转发的数据包被称为数据报（datagram）。帧（frame）是数据链路层单元，分段（segment）是传输层单元，消息（message）是高层单元，均不符合网络层定义。

# 20. 单选题

原题：
Transport-layer packets are called:
A. message
B. segment
C. datagram
D. frame

答案：B
解析：传输层的数据包被称为分段（segment）。数据报是网络层单元，帧是数据链路层单元，消息是应用层等高层单元，均与传输层无关。

# 21. 单选题

原题：
The units of data exchanged by a link-layer protocol are called ().
A. Frames
B. Segments
C. Datagrams
D. bit streams

答案：A
解析：链路层（数据链路层）协议交换的数据单元是帧（Frame）。

  • B（Segments，分段）是传输层数据单元；
  • C（Datagrams，数据报）是网络层数据单元；
  • D（bit streams，比特流）是物理层数据单元。

# 22. 单选题

原题：
There are two fundamental approaches to building a network core, () and packet switching.
A. electrical current switching
B. circuit switching
C. data switching
D. message switching

答案：B
解析：构建网络核心的两种基本方式是电路交换（circuit switching） 和分组交换（packet switching）。

  • A（电流交换）、C（数据交换）不是标准网络术语；
  • D（报文交换）是分组交换的前身，不属于网络核心的两大基础分类。

# 23. 单选题

原题：
Datagram networks and virtual-circuit networks differ in that ().
A. datagram networks are circuit-switched networks, and virtual-circuit networks are packet-switched networks.
B. datagram networks are packet-switched networks, and virtual-circuit networks are circuit-switched networks.
C. datagram networks use destination addresses and virtual-circuit networks use VC. numbers to forward packets toward their destination.
D. datagram networks use VC. numbers and virtual-circuit networks use destination addresses to forward packets toward their destination.

答案：C
解析：数据报网络与虚电路网络的核心区别在于转发依据：

  • 数据报网络（如 IP 网络）使用 ** 目的地址（destination addresses）** 转发数据包；
  • 虚电路网络（如 ATM、帧中继）使用 ** 虚电路号（VC numbers）** 转发数据包。
    A、B 错误，二者均属于分组交换网络；D 颠倒了二者的转发依据。

# 24. 单选题

原题：
() means that the switch must receive the entire packet before it can begin to transmit the first bit of the packet onto the outbound link.
A. Store-and-forward transmission
B. FDM
C. End-to-end connection
D. TDM

答案：A
解析：存储 - 转发传输（Store-and-forward transmission） 的定义就是交换机必须完整接收整个数据包后，才能开始向出站链路发送第一个比特。

  • B（FDM，频分复用）、D（TDM，时分复用）是电路交换的多路复用技术；
  • C（端到端连接）是通信模式，与该定义无关。

# 25. 单选题

原题：
In () networks, the resources needed along a path to provide for communication between the end system are reserved for the duration of the communication session.
A. packet-switched
B. data-switched
C. circuit-switched
D. message-switched

答案：C
解析：电路交换（circuit-switched）网络的核心特征是：在通信会话期间，路径上的资源（带宽、时隙等）被预分配、独占保留，直到会话结束。

  • A（分组交换）、D（报文交换）均为按需分配资源，不做预保留；
  • B（数据交换）不是标准网络术语。

# 26. 单选题

原题：
In () networks, the resources are not reserved; a session's messages use the resources on demand, and as a consequence, may have to wait for access to communication link.
A. packet-switched
B. data-switched
C. circuit-switched
D. message-switched

答案：A
解析：分组交换（packet-switched） 网络不预分配资源，消息按需占用链路资源，因此可能需要排队等待链路访问。

  • C（电路交换）是预分配资源，无需等待；
  • B（数据交换）不是标准术语；
  • D（报文交换）虽也按需分配，但分组交换是现代互联网的核心技术，是本题的标准答案。

# 27. 单选题

原题：
In a circuit-switched network, if each link has n circuits, for each link used by the end-to-end connection, the connection gets () of the link's bandwidth for the duration of the connection.
A. a fraction 1/n
B. all
C. 1/2
D. n times

答案：A
解析：在电路交换网络中，若一条链路有 n 个电路，每个端到端连接会独占该链路 1/n 的带宽，直到连接结束。

  • B（全部）是单电路链路的情况；
  • C（1/2）、D（n 倍）不符合电路交换的带宽分配逻辑。

# 28. 单选题

原题：
For (), the transmission rate of a circuit is equal to the frame rate multiplied by the number of bits in a slot.
A. CDMA
B. packet-switched network
C. TDM
D. FDM

答案：C
解析：TDM（时分复用） 的传输速率计算公式为： 电路传输速率 = 帧速率 × 时隙比特数 ，这是 TDM 的核心定义。

  • A（CDMA，码分复用）是按编码区分信道；
  • B（分组交换网络）无固定帧 / 时隙结构；
  • D（FDM，频分复用）是按频率划分带宽，与帧 / 时隙无关。

# 29. 单选题

原题：
The network that forwards packets according to host destination addresses is called () network.
A. circuit-switched
B. packet-switched
C. virtual-circuit
D. datagram

答案：D
解析：数据报（datagram） 网络的核心特征是：根据主机的 目的地址 转发数据包（如 IP 网络）。

  • A（电路交换）不按地址转发；
  • B（分组交换）是大类，包含数据报和虚电路两种子类型，不是本题的精准答案；
  • C（虚电路）按虚电路号转发，而非目的地址。

# 30. 单选题

原题：
The network that forwards packets according to virtual-circuit numbers is called () network.
A. circuit-switched
B. packet-switched
C. virtual-circuit
D. datagram

答案：C
解析：虚电路（virtual-circuit） 网络的核心特征是：根据 虚电路号（VC numbers） 转发数据包。

  • A（电路交换）无虚电路号概念；
  • B（分组交换）是大类，不是精准答案；
  • D（数据报）按目的地址转发，与虚电路号无关。

# 31. 单选题

原题：
The time required to propagate from the beginning of the link to the next router is ().
A. queuing delay
B. processing delay
C. propagation delay
D. transmission delay

答案：C
解析：传播时延（propagation delay） 的定义就是信号从链路起点传播到下一个路由器所需的时间。

  • A（排队时延）是分组在路由器队列中等待转发的时间；
  • B（处理时延）是路由器处理分组的时间；
  • D（传输时延）是将分组完整发送到链路上的时间。

# 32. 单选题

原题：
Processing delay does not include the time to ().
A. examine the packet's header
B. wait to transmit the packet onto the link
C. determine where to direct the packet
D. check bit-error in the packet

答案：B
解析：处理时延包含检查分组首部、检测比特错误、确定转发路径等操作的时间，不包含等待发送到链路的时间（该时间属于排队时延）。

# 33. 单选题

原题：
In the following four descriptions, which one is correct? ()
A. The traffic intensity must be greater than 1.
B. The fraction of lost packets increases as the traffic intensity decreases.
C. If the traffic intensity is close to zero, the average queuing delay will be close to zero.
D. If the traffic intensity is close to one, the average queuing delay will be close to one.

答案：C
解析：

  • A 错误：流量强度（traffic intensity）通常小于 1，大于 1 会导致队列无限增长；
  • B 错误：流量强度越低，丢包率越低；
  • C 正确：流量强度趋近于 0 时，几乎无排队，平均排队时延趋近于 0；
  • D 错误：流量强度趋近于 1 时，平均排队时延会急剧增大，而非趋近于 1。

# 34. 单选题

原题：
Suppose, a is the average rate at which packets arrive at the queue, R is the transmission rate, and all packets consist of L bits, then the traffic intensity is (),
A. LR/a
B. La/R
C. Ra/L

答案：B
解析：流量强度的计算公式为： 流量强度 = (分组长度 × 分组到达率) / 链路传输率 ，即 La/R 。

  • 推导：分组到达率为 a （个 / 秒），每个分组长度 L （比特），则链路的输入比特率为 La （比特 / 秒），链路传输率为 R （比特 / 秒），因此流量强度为 La/R 。

# 35. 单选题

原题：
Suppose there is exactly one packet switch between a sending host and a receiving host. The transmission rates between the sending host and the switch and between the switch and the receiving host are R1 and R2, respectively. Assuming that the switch uses store-and-forward packet switching, what is the total end-to-end delay to send a packet of length L? (Ignore queuing delay, propagation delay, and processing delay.) ()
A. L/R1+L/R2
B. L/R1
C. L/R2
D. none of the above

答案：A
解析：存储 - 转发交换中，分组需要先完整发送到交换机（时延 L/R1 ），再由交换机完整发送到接收主机（时延 L/R2 ），总时延为两段传输时延之和 L/R1 + L/R2 。

# 36. 单选题

原题：
We are sending a 30 Mbit MP3 file from a source host to a destination host. Suppose there is only one link between source and destination and the link has a transmission rate of 10 Mbps. Assume that the propagation speed is 2 * 10^8 meters/sec, and the distance between source and destination is 10,000 km. Also suppose that message switching is used, with the message consisting of the entire MP3 file. How many bits will the source have transmitted when the first bit arrives at the destination?
A. 1 bit
B. 30,000,000 bits
C. 500,000 bits
D. none of the above

答案：C
解析：

  1. 先计算传播时延： 传播时延 = 距离 / 传播速度 = 10,000×10^3 m / (2×10^8 m/s) = 0.05 s
  2. 源主机在 0.05 秒内发送的比特数： 发送比特数 = 传输率 × 传播时延 = 10×10^6 bps × 0.05 s = 500,000 bits
    因此当第一个比特到达目的地时，源主机已发送 500,000 比特。

In the following entries, which is not a kind of access network?()

# 38. 单选题

原题：
The following technologies may be used for residential access, except
A. HFC
B. DSL
C. Dial-up modem
D. FDDI

答案：D
解析：

  • A（HFC，混合光纤同轴网）、B（DSL，数字用户线）、C（拨号调制解调器）均为住宅接入技术；
  • D（FDDI，光纤分布式数据接口）是用于企业局域网的高速光纤技术，不用于住宅接入。

# 39. 单选题

原题：
Which kind of media is not a guided media? ()
A. twisted-pair copper wire
B. a coaxial cable
C. fiber optics
D. digital satellite channel

答案：D
解析：导引型媒体（guided media） 指信号沿物理介质传播，包括双绞线、同轴电缆、光纤；非导引型媒体 指信号在自由空间传播，数字卫星信道属于非导引型媒体。

# 40. 单选题

原题：
在 OSI 参考模型中，直接为会话层提供服务的是____。
A. 应用层
B. 表示层
C. 传输层
D. 网络层

答案：C
解析：OSI 七层模型从下到上为：物理层→数据链路层→网络层→传输层→会话层→表示层→应用层。下层为上层提供服务，因此传输层直接为会话层提供服务。

# 41. 单选题

原题：
在 OSI 参考模型中，下列功能需由应用层的相邻层实现的是____。
A. 对话管理
B. 数据格式转换
C. 路由选择
D. 可靠数据传输

答案：B
解析：OSI 七层模型从下到上为：物理层→数据链路层→网络层→传输层→会话层→表示层→应用层。应用层的直接相邻层是表示层，其核心功能就是数据格式转换（如加密、压缩、编码转换等）。

  • A（对话管理）是会话层的功能；
  • C（路由选择）是网络层的功能；
  • D（可靠数据传输）是传输层的功能。

# 42. 单选题

原题：
TCP/IP 参考模型的网络层提供的是____。
A. 无连接不可靠的数据报服务
B. 无连接可靠的数据报服务
C. 有连接不可靠的虚电路服务
D. 有连接可靠的虚电路服务

答案：A
解析：TCP/IP 模型的网络层核心协议是 IP 协议，IP 协议是无连接、不可靠的，它只负责尽最大努力交付数据报，不保证数据的顺序、完整性和可达性，因此网络层提供的是无连接不可靠的数据报服务。

  • 可靠、有连接的服务由传输层的 TCP 协议实现，而非网络层。

# 43. 单选题

原题：
下列选项中，不属于网络体系结构中所描述的内容是____。
A. 网络的层次
B. 每一层使用的协议
C. 协议的内部实现细节
D. 每一层必须完成的功能

答案：C
解析：网络体系结构是对网络层次、各层功能、层间接口和协议的抽象定义，不包含协议的具体内部实现细节（实现细节由厂商自主完成，属于 “如何实现”，而非体系结构定义的 “做什么”）。

  • A、B、D 均为网络体系结构的核心描述内容。

# 44. 单选题

原题：
在 OSI 参考模型中，自下而上第一个提供端到端服务的层次是____。
A. 数据链路层
B. 传输层
C. 会话层
D. 应用层

答案：B
解析：OSI 模型中，传输层是自下而上第一个为端系统（源主机到目的主机）提供端到端通信服务的层次。

  • 数据链路层、网络层仅提供点到点（相邻节点间）的服务；
  • 会话层、应用层基于传输层的端到端服务工作，并非第一个提供该服务的层次。

# 45. 单选题

原题：
假设 OSI 参考模型的应用层欲发送 400 B 的数据 (无拆分), 除物理层和应用层之外，其他各层在封装 PDU 时均引入 20 B 的额外开销，则应用层数据传输效率约为____
A. 80%
B. 83%
C. 87%
D. 91%

答案：A
解析：

  1. 计算开销层数：OSI 共 7 层，排除物理层和应用层，剩余 5 层（数据链路层、网络层、传输层、会话层、表示层），每层加 20B 开销，总开销 = 5 × 20B = 100B
  2. 计算总传输数据量：应用层数据 400B + 总开销 100B = 500B
  3. 计算传输效率： (有效数据 / 总数据) × 100% = (400 / 500) × 100% = 80%

# 计算机网络基础 45 题核心知识点完整清单

# 一、网络体系结构与 OSI/TCP/IP 模型（核心考点）

# 1. OSI 七层模型（自下而上）

层级核心功能相邻层服务关系对应 PDU关键考点
物理层比特流传输、电气 / 机械 / 功能特性为数据链路层提供服务比特（bit）导引型 / 非导引型媒体分类、传输时延 / 传播时延区分
数据链路层帧封装、差错检测（CRC）、MAC 寻址、流量控制为网络层提供服务帧（Frame）链路层数据单元、差错检测功能归属
网络层路由选择、IP 寻址、拥塞控制、分组转发为传输层提供服务数据报（Datagram）数据报网络 / 虚电路网络转发依据、路由选择功能归属
传输层端到端通信、可靠传输（TCP）/ 不可靠传输（UDP）、端口寻址为会话层提供服务分段（Segment）自下而上第一个提供端到端服务的层次、可靠数据传输功能归属
会话层会话管理、同步、断点续传、对话管理为表示层提供服务消息（Message）对话管理功能归属、为会话层提供服务的层次
表示层数据格式转换、加密、压缩、编码转换为应用层提供服务消息（Message）应用层相邻层、数据格式转换功能归属
应用层面向用户的应用（HTTP、FTP、DNS 等）调用表示层服务消息（Message）应用层相邻层功能、应用层数据传输效率计算

# 2. TCP/IP 四层模型

层级对应 OSI 层级核心协议服务特性关键考点
应用层会话层 + 表示层 + 应用层HTTP、FTP、DNS、SMTP面向应用应用层协议分类
传输层传输层TCP、UDP端到端、可靠（TCP）/ 不可靠（UDP）TCP/UDP 服务特性区分
网络层网络层IP、ICMP、ARP无连接、不可靠、尽最大努力交付网络层服务特性、IP 协议核心特点
网络接口层物理层 + 数据链路层以太网、PPP点到点传输链路层协议分类

# 3. 网络体系结构核心定义

  • 体系结构描述内容：网络层次、各层功能、层间接口、各层使用的协议（不包含协议内部实现细节）
  • 层间服务原则：下层为上层提供服务，上层调用下层服务，服务访问点（SAP）是层间交互的接口
  • 关键考点：体系结构不包含的内容、层间服务关系

# 二、网络核心技术：交换与复用

# 1. 网络核心两大基础交换方式

交换方式资源分配转发依据典型代表关键特性
电路交换预分配、独占资源电路时隙传统电话网通信期间资源全程保留、无排队时延、带宽固定分配（1/n 链路带宽）
分组交换按需分配、共享资源目的地址（数据报）/ 虚电路号（虚电路）互联网（IP 网络）资源不预留、可能排队等待、统计复用、存储 - 转发机制

# 2. 分组交换的两种子类型

网络类型转发依据连接特性典型代表关键考点
数据报网络主机目的 IP 地址无连接互联网（IP）按目的地址转发、无连接不可靠服务
虚电路网络虚电路号（VC 号）有连接ATM、帧中继按 VC 号转发、面向连接、分组交换的一种

# 3. 多路复用技术（电路交换核心技术）

复用方式核心原理传输速率计算关键考点
时分复用（TDM）按时间划分时隙，每个电路占用固定时隙电路传输速率 = 帧速率 × 时隙比特数TDM 速率公式、与 FDM 的区分
频分复用（FDM）按频率划分带宽，每个电路占用固定频段频段带宽之和 = 链路总带宽FDM 与 TDM 的应用场景区分
码分复用（CDMA）按编码区分信道，所有用户共享同一频段编码正交性CDMA 的核心特点

# 4. 存储 - 转发传输

  • 定义：交换机必须完整接收整个数据包后，才能开始向出站链路发送第一个比特
  • 总时延计算：多段链路总时延 = 各段链路传输时延之和（ L/R1 + L/R2 + ... ）
  • 关键考点：存储 - 转发的定义、单交换机场景下的端到端时延计算

# 三、网络时延与流量强度（计算核心考点）

# 1. 四大网络时延

时延类型定义计算公式关键区分
传输时延（发送时延）将分组完整发送到链路的时间L/R （L = 分组长度，R = 链路传输率）与链路长度无关，仅与分组长度和传输率有关
传播时延信号在链路中从起点传播到终点的时间d/s （d = 链路长度，s = 信号传播速度）与分组长度无关，仅与链路长度和传播速度有关
处理时延路由器处理分组的时间（检查首部、差错检测、路由决策）硬件相关，通常忽略不包含排队等待发送的时间（排队时延）
排队时延分组在路由器输出队列中等待转发的时间与流量强度正相关流量强度越接近 1，排队时延急剧增大

# 2. 流量强度

  • 定义：链路输入比特率与链路传输率的比值，反映链路拥塞程度
  • 计算公式： 流量强度 = (分组长度L × 分组到达率a) / 链路传输率R = La/R
  • 关键特性：
    • 流量强度 < 1：队列稳定，平均排队时延有限
    • 流量强度 → 0：平均排队时延 → 0
    • 流量强度 → 1：平均排队时延急剧增大
    • 流量强度 > 1：队列无限增长，丢包率急剧上升
  • 关键考点：流量强度公式、流量强度与排队时延 / 丢包率的关系

# 3. 经典计算题型

  • 单链路报文交换发送比特数计算：先算传播时延，再算传播时延内发送的比特数（如 36 题： 10^7 bps × 0.05s = 500,000 bits ）
  • 应用层数据传输效率计算： 传输效率 = 有效数据量 / (有效数据量 + 总开销) × 100% （如 45 题： 400/(400+5×20)=80% ）
  • 存储 - 转发端到端时延计算：多段链路传输时延之和（如 35 题： L/R1 + L/R2 ）

# 四、接入网与传输媒体

# 1. 接入网分类

  • 标准分类：住宅接入（Residential Access）、企业接入（Company Access）、无线接入（Wireless Access）
  • 非标准分类：本地接入（Local Access，不属于标准接入网类型）
  • 住宅接入技术：
    • DSL（数字用户线）：利用现有电话线
    • HFC（混合光纤同轴网）：利用有线电视同轴电缆
    • 拨号调制解调器（Dial-up Modem）：早期低速接入
    • 非住宅接入技术：FDDI（光纤分布式数据接口，企业局域网技术）
  • 关键考点：接入网分类、住宅接入技术与企业接入技术的区分

# 2. 传输媒体分类

媒体类型定义典型代表关键考点
导引型媒体（Guided Media）信号沿物理介质定向传播双绞线、同轴电缆、光纤导引型媒体的判断、与非导引型的区分
非导引型媒体（Unguided Media）信号在自由空间传播无线电波、微波、卫星信道数字卫星信道属于非导引型媒体

# 五、易错题与高频陷阱汇总

# 1. 易混淆概念区分

易混淆点正确区分错误陷阱
分组交换 vs 数据报网络分组交换是大类，包含数据报和虚电路两种；数据报是分组交换的子类型误将分组交换等同于数据报网络（29 题陷阱）
传输时延 vs 传播时延传输时延是发送分组的时间，传播时延是信号传播的时间混淆两者的计算公式和影响因素
处理时延 vs 排队时延处理时延是路由器处理分组的时间，排队时延是等待发送的时间误将排队时延计入处理时延（32 题考点）
虚电路网络 vs 电路交换虚电路是分组交换的一种，电路交换是独立的交换方式误将虚电路等同于电路交换
TCP/IP 网络层 vs 传输层服务网络层是无连接不可靠，传输层 TCP 是有连接可靠误将网络层认为是可靠服务（42 题考点）

# 2. 高频计算陷阱

  • OSI 开销层数计算：排除物理层和应用层，剩余 5 层（数据链路层、网络层、传输层、会话层、表示层），每层 20B，总开销 100B（45 题）
  • 流量强度公式： La/R ，注意分组到达率 a 的单位（个 / 秒），分组长度 L 的单位（比特）（34 题）
  • 存储 - 转发时延：必须完整接收分组才能转发，因此总时延是各段传输时延之和，而非仅一段（35 题）
  • 传播时延内发送比特数：先算传播时延，再用传输率 × 传播时延，而非用文件大小计算（36 题）

# 3. 层级功能归属陷阱

  • 应用层相邻层：表示层（功能：数据格式转换），而非会话层（41 题）
  • 为会话层提供服务的层次：传输层（40 题）
  • 自下而上第一个提供端到端服务的层次：传输层（44 题）
  • 差错检测功能归属：数据链路层（16 题）
  • 路由选择功能归属：网络层（41 题干扰项）
  • 对话管理功能归属：会话层（41 题干扰项）
