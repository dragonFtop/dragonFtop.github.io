# 【单调栈】柱状图中的最大矩形

# 题目

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。

示例 1:

输入：heights = [2,1,5,6,2,3]
输出：10
解释：最大的矩形为图中红色区域，面积为 10

示例 2：

输入： heights = [2,4]
输出： 4

提示：
1 <= heights.length <=105
0 <= heights[i] <= 104

# 解答

# 单调栈 - 双躺找左右边界 - 1

我个人理解：找左右边界可以从左往右找右边第一个更矮的柱子，也可以从左往右找左边第一个更矮的柱子（从右往左亦然有两种写法）。先写一个从左往右找左边第一个更矮的柱子这种方式：


这里只需要说从左往右的情况就行了，这里的栈用来存储当前元素左侧可能的左边界的索引（栈中索引对应的值一定是单调递增的）。
需要判断的是遍历到当前的元素比栈顶的索引对应的值小时，栈顶的元素就不能继续作为后序遍历的元素的左边界了，也就是栈顶元素被淘汰了，需要弹出。
每次遍历都将当前元素入栈，作为后续遍历的候选左边界。
逆向求元素右边界的方式同理，从右往左找，找到的元素索引就是当前元素有边界的索引。

在这个代码中，每个元素的左边界初始化为 - 1，右边界初始化为 n。
我从左往右遍历的过程中，栈用来存储还没有找到右边界元素的索引（栈中索引对应的值一定是单调递增的）。
每当遍历到一个元素的时候，我们检查这个元素是否是栈中元素的右边界，也就是如果当前元素的值比栈顶元素小（相等也不行），当前元素就是栈顶元素的右边界，栈顶元素找到了右边界，可以弹出了。
每次遍历到一个元素，我们都将这个元素入栈，让其在栈中等待找到右边界。
