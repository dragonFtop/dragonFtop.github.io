# 【分组密码】概念性题目整理

# 单选题

# 第 1 题

题目：理论上不能破译的密码算法是 ( )
A、AES 密码算法
B、RSA 密码算法
C、一次一密密码算法 (OTP)
D、非对称加密算法
答案：C
解析：一次一密（OTP）是目前唯一信息论安全、理论上绝对不可破译的密码，只要满足密钥真随机、仅使用一次、长度≥明文，无限算力也无法破解；AES、RSA 等算法均属于计算安全，理论上存在被穷举破解的可能。

# 第 2 题

题目：DES 算法密钥为 64 位，其密钥空间为 ( )。
A、64
B、2642^{64}264
C、56
D、2562^{56}256
答案：D
解析：DES 名义密钥长度 64bit，其中 8bit 为奇偶校验位，有效密钥长度仅 56bit，因此密钥空间大小为2562^{56}256。

# 第 3 题

题目：Feistel 密码结构是对称分组密码中的一种典型结构，它 ()。
A、一轮循环可对整个分组进行更新
B、一轮循环仅对半个分组进行更新
C、要求其中的 F 函数可逆
D、也是 RC5 密码算法的基本结构
答案：B
解析：Feistel 结构将分组分为左右两半，单轮运算只会修改右半分组，左半分组直接保留；F 函数不需要可逆；RC5 不属于 Feistel 结构密码。

# 第 4 题

题目：DES 算法中的 8 个 S 盒子是该密码算法重要而唯一的 ( ) 组件。
A、非线性
B、线性
C、加密
D、Feistel
答案：A
解析：DES 整体大部分操作均为线性变换，仅 S 盒引入非线性混淆能力，是整个算法抵抗线性密码分析、保证安全性的核心关键。

# 第 5 题

题目：对于分组密码的操作方式，NIST 规定了四种标准的操作模式，它们是 ()。
A、DES、IDEA、RSA 和 RC5
B、ECB、CBC、CFB 和 OFB
C、AES、RSA、CFB 和 OFB
D、IDEA、CRC、MD5 和 HASH
答案：B
解析：NIST 规定的分组密码四大经典工作模式为 ECB、CBC、CFB、OFB；其余选项均为加密算法、哈希函数，不属于工作模式。

# 第 6 题

题目：国际数据加密标准 IDEA 的三种基本运算是 ( )。
A、逐位异或、循环移位和置换
B、逐位异或、模 65536 加法和模 65537 乘法
C、逐位异或、置换和乘法运算
D、逐位异或、模 655536 的加法和乘法运算
答案：B
解析：IDEA 算法全程仅使用三种基础运算：按位异或、模2162^{16}216（65536）整数加法、模 65537 整数乘法。

# 第 7 题

题目：对称密码算法 AES 的分组长度和密钥长度分别为 ()。
A、分组长度 64bit, 密钥长度为 128bit
B、分组长度 128bit, 密钥长度为 192bit
C、分组长度 128bit, 密钥长度可为 128,192,256bit
D、分组长度 256bit, 密钥长度可为 128,192,256bit
答案：C
解析：AES 固定分组长度为 128bit，支持三种可选密钥规格：128bit、192bit、256bit，对应 AES-128、AES-192、AES-256 三个版本。

# 第 8 题

题目：相当于 DES 中的 S 盒，为 AES 算法提供非线性变换的是 ()。
A、字节替代
B、列混合变换
C、轮密钥加法
D、行位移变换
答案：A
解析：AES 的字节替代（S 盒变换） 是算法中唯一的非线性混淆步骤，作用和 DES 的 S 盒完全一致，用来打乱线性规律、抵抗密码分析。

# 第 9 题

题目：下列哪些是对称密码算法 AES 的基本变换。()
①字节替代 ②列混合变换 ③半交换 ④置换 ⑤行位移变换
A、①②③④
B、①②③⑤
C、①②⑤
D、②③④
答案：C
解析：AES 标准 4 轮基础变换为：字节替代、行位移、列混合、轮密钥加；题干中①②⑤均属于 AES 核心变换，半交换、置换不属于。

# 第 10 题

题目：下列 () 不是对称密码算法 AES 的基本变换。
A、字节替代
B、Feistel 变换
C、行位移变换
D、列混合变换
答案：B
解析：AES 属于 SP 网络结构分组密码，全程不使用 Feistel 变换；Feistel 是 DES 等老式分组密码的经典结构。

# 多选题

# 第 11 题

题目：下面对 Feistel 网络的描述正确的是：()
A、Feistel 一种代换网络
B、输出的每一个比特的密文都和输入的明文及密钥的各比特相关
C、加密过程的算法复杂度比解密过程的算法的复杂度高很多
D、Feistel 网络中每轮的结构都相同
答案：ABD
解析：

  • A✅：Feistel 是典型的代换 - 置换网络变体
  • B✅：经过多轮迭代后，实现雪崩效应，密文每比特都和明文、密钥全部比特关联
  • C❌：Feistel 加密和解密仅轮密钥顺序相反，复杂度几乎完全一致
  • D✅：Feistel 网络每一轮的运算结构完全相同，仅子密钥不同

# 第 12 题

题目：下面关于分组密码的要求，正确的是：()
A、分组长度足够大
B、密钥空间足够大
C、密码算法足够复杂
D、系统的安全性与算法的保密性有关
答案：ABC
解析：

  • A/B/C✅：分组长度大防重复、密钥空间大防穷举、算法复杂抗分析，是分组密码三大基础设计要求
  • D❌：遵循柯克霍夫斯原则，现代密码安全性仅依赖密钥保密，算法必须公开

# 第 13 题

题目：下列是 AES 的基本变换的是 ():
A、字节替代
B、列混合
C、行移位
D、密钥加
答案：ABCD
解析：AES 完整的四大轮变换：字节替代 (SubBytes)、行移位 (ShiftRows)、列混合 (MixColumns)、轮密钥加 (AddRoundKey)，四个选项全部正确。

# 第 14 题

题目：下列密钥长度中，满足 AES 算法要求的是 ():
A、160bit
B、192bit
C、64bit
D、256bit
答案：BD
解析：AES 官方仅支持 128bit、192bit、256bit 三种密钥长度，因此 192bit、256bit 符合要求。

# 第 15 题

题目：下列说法不正确的是 ():
A、IDEA 算法的三种基本运算中，任意两个运算都不满足分配律和结合律
B、AES 算法的字节代替运算关于字节是线性变换
C、二重 DES 可以有效抵御中间相遇攻击
D、AES 算法理论上可以破译
答案：BC
解析：

  • A 正确
  • B 错误：字节替代是非线性变换，不是线性变换
  • C 错误：二重 DES 完全无法抵御中间相遇攻击，只有三重 DES 才有明显安全提升
  • D 正确：AES 属于计算安全，理论上可暴力穷举破译
    题干要求选不正确的选项，因此答案为 BC。

# 判断题

# 第 16 题

题目：DES 的加密算法与解密算法不相同。
答案：错
解析：DES 是 Feistel 结构密码，加密和解密流程完全相同，仅解密时子密钥的使用顺序与加密相反，整体算法结构一致。

# 第 17 题

题目：DES 算法的初始置换和逆初始置换是的非线性变换。
答案：错
解析：初始置换、逆初始置换仅单纯打乱比特顺序，全程为纯线性位置重排，不具备非线性特性。

# 第 18 题

题目：DES 算法的密钥长度为 64 位，故密钥空间为2642^{64}264。
答案：错
解析：64 位密钥中有 8 位为校验位，有效密钥仅 56 位，真实密钥空间为2562^{56}256，不是2642^{64}264。

# 第 19 题

题目：AES 的加密函数与解密函数相同。
答案：错
解析：AES 不是 Feistel 结构，加密和解密的变换步骤、逆变换流程不同，加密函数与解密函数并不相同。

# 第 20 题

题目：分组密码的 CBC 模式的主要缺点是在相同密钥的条件下，相同的明文分组所产生的密文分组相同。
答案：错
解析：描述的是 ECB 电子密码本模式的致命缺点；CBC 模式引入了分组链式异或，相同明文分组也会产出不同密文分组，安全性远高于 ECB。

# 第 21 题

题目：DES 算法的非线性变换有 S 变换和置换。
答案：错
解析：DES 中只有 S 盒（S 变换）是非线性组件，其余所有置换、移位操作均为纯线性变换。

# 第 22 题

题目：实现扩散和混淆是现代分组密码设计的基本原则。
答案：对
解析：扩散（打乱明文统计规律）、混淆（密钥与密文关系复杂化），是香农提出的密码设计两大核心黄金原则，所有现代分组密码均严格遵循。

# 第 23 题

题目：分组密码 DES 中的 P 置换主要是起到混淆作用。
答案：错
解析：DES 的 P 置换核心作用是扩散，将 S 盒输出的 1 比特扩散到多比特，让变化快速传遍整个分组，而非混淆。

# 第 24 题

题目：分组密码 AES 中的字节替代是算法中唯一的非线性变换。
答案：对
解析：AES 的行移位、列混合、轮密钥加均为线性运算，仅字节替代步骤引入非线性，是算法安全的核心。

# 第 25 题

题目：分组密码算法 SM4 的分组长度和密钥长度都是 128bit。
答案：对
解析：SM4 是我国商用国密对称分组标准，固定分组长度 128bit、密钥长度 128bit，符合题目描述。
