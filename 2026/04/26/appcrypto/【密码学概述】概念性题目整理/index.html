# 【密码学概述】概念性题目整理

# 单选题

# 第 1 题

题目：常规加密技术通常是指 ( )。
A、隐写术
B、保密通信
C、对称加密技术
D、非对称加密技术
答案：C
解析：常规加密也叫对称密钥加密、单密钥加密，通信双方使用相同的密钥完成加解密，是最早、最传统的加密技术，因此常规加密技术特指对称加密技术。

# 第 2 题

题目：以下关于对称密码体制的描述中，错误的是 ( )
A、对称密码体制的加密密钥和解密密钥相同或可相互导出。
B、对称密码体制是指加密算法和解密算法相同的密码算法。
C、加解密处理速度快是对称密钥密码体制的优点之一。
D、对称密码体制的密钥管理较非对称密码更困难。
答案：B
解析：对称密码的核心定义是密钥相同 / 可互推，而非算法一定相同；B 选项概念表述错误。对称算法计算量小、速度快；同时需要提前分发共享密钥，大规模场景下密钥分发与管理远难于非对称密码。

# 第 3 题

题目：以下关于公开密钥密码体制的描述中，错误的是 ( )
A、加解密处理代价较常规密钥密码体制大，一般不直接用于对数据文件的加密。
B、密码算法加解密的密钥不同，解密密钥仍要求保密，但加密密钥应公开。
C、公开密钥密码就是密钥可以公开的密码体制。
D、公开密钥密码体制较常规密钥密码体制有更好的开放性。
答案：C
解析：公开密钥密码（非对称加密）是一对密钥：公钥公开、私钥严格保密，不是所有密钥都可以公开。非对称加密运算复杂度高，通常只用来加密对称密钥、数字签名，不会直接加密大体积文件；相比对称密码，无需提前共享秘密密钥，开放性、扩展性更强。

# 第 4 题

题目：密码学中，受限制的密码算法是指 ( )。
A、不安全的算法
B、其安全性依赖于对算法的保密
C、受专利法保护，不能公开使用的算法
D、其安全性依赖于对密钥的保密
答案：B
解析：受限制算法的安全根基是隐藏算法本身细节，一旦算法泄露就彻底失去安全性；现代密码学遵循柯克霍夫斯原则：算法公开，安全性仅依赖密钥保密。

# 第 5 题

题目：以下对密码系统攻击目标的描述中，正确的是 ( )。
A、破译出解密算法的细节
B、在不知道密钥的前提下，破译或部分破译加密信息，甚至密钥。
C、破译出加密算法的细节
D、破译出加解密算法
答案：B
解析：现代密码攻击默认算法完全公开，攻击者目标是未知密钥的情况下，还原明文、推算密钥，而非破解公开的算法本身。

# 第 6 题

题目：密码学的两个分支是 ( ) 和密码分析学。其中前者是对信息进行编码以保护信息的一门学问，后者是研究分析破译密码的学问。
A、密码协议
B、密码设计学
C、密码应用
D、密码编码学
答案：D
解析：密码学分为两大基础分支：密码编码学（研究如何设计加密方案、保护信息）、密码分析学（研究如何破解密码、还原信息）。

# 第 7 题

题目：密码分析通常是指 ( )。
A、对加密与解密算法的构造与设计
B、对加密信息的攻击 (破译) 过程
C、如何使消息保密的技术
D、对信息的编码分析过程
答案：B
解析：密码分析就是密码破译，指攻击者针对密文、加密系统开展破解、推导明文或密钥的攻击行为。

# 第 8 题

题目：评估密码系统安全性的主要方法不包括 ( )。
A、无条件安全性
B、计算安全性
C、密钥安全性
D、可证明安全性
答案：C
解析：密码安全的三类主流评估标准：

  1. 无条件安全（信息论安全，无限算力也无法破解）
  2. 计算安全（现有算力、时间条件下无法有效破解）
  3. 可证明安全（可数学规约证明安全强度）
    密钥安全性不属于独立的系统安全评估分类。

# 第 9 题

题目：1949 年 () 发表了 “保密系统的通信理论” 一文，这篇文章奠定了密码学的理论基础，推动着密码学从艺术向科学的转变。
A、Hellman
B、Shamir
C、Shannon
D、Diffie
答案：C
解析：克劳德・香农（Shannon）1949 年发表该著作，用信息论系统化定义密码安全，让密码学从经验技巧正式变为严谨科学。Diffie、Hellman、Shamir 是非对称加密领域的代表人物。

# 第 10 题

题目：以下对密码攻击者假设的描述，错误的是 ( )。
A、可以拥有目前最先进的计算资源
B、可以不惜成本
C、可以拥有足够的时间
D、不知道密码算法的细节
答案：D
解析：密码学默认柯克霍夫斯假设：攻击者完全知晓完整的密码算法细节，仅密钥保密；同时默认攻击者具备顶级算力、充足时间与成本。

# 判断题

# 第 11 题

题目：不可能存在理论上可确保安全的密码体制
答案：错
解析：存在理论上绝对安全（无条件安全）的密码，典型如一次一密（One-Time Pad），只要密钥满足真随机、长度≥明文、只用一次，就无法被攻破。

# 第 12 题

题目：现代密码体制把算法和密钥分开，只需要保证密钥的保密性就行了，算法是可以公开的
答案：对
解析：柯克霍夫斯原则核心：算法公开、标准化公开审查，所有安全仅依靠密钥保密，大幅提升密码系统的实际安全水平。

# 第 13 题

题目：尽管没有信息加密的密钥，但只要知道加密算法的细节就可以对信息进行解密
答案：错
解析：现代密码设计保证：公开算法、无密钥的情况下，无法从密文还原明文。

# 第 14 题

题目：对称密码算法是指加密算法和解密算法相同的密码算法
答案：错
解析：对称密码的定义是加解密使用的密钥相同、或可互相推导，算法本身不一定完全相同。

# 第 15 题

题目：单密钥密码算法是指仅有加密密钥的密码算法
答案：错
解析：单密钥（对称密码）是同一个密钥既用于加密、也用于解密，不是只有加密密钥。

# 第 16 题

题目：公开密钥密码就是密钥可以公开的密码体制
答案：错
解析：仅公钥对外公开，对应的私钥必须严格保密、绝对不能公开，并非全部密钥都可公开。

# 第 17 题

题目：密码学包括两大分支，即密码编码学和密码分析学
答案：对
解析：密码学标准两大分支：编码学（造密码、保信息）、分析学（破密码、攻系统）。

# 第 18 题

题目：一个密码系统的保密性不能依赖于对加密体制或算法的保密，而仅依赖于密钥的安全性
答案：对
解析：符合柯克霍夫斯设计原则，依赖算法保密的算法安全性极差，一旦算法泄露系统就彻底失效。

# 第 19 题

题目：密钥少，便于管理是常规密码算法的优点之一
答案：错
解析：常规对称密码的缺点就是密钥分发、管理难度大；用户数量增多时，密钥数量爆炸式增长，管理极其复杂。

# 第 20 题

题目：加解密处理速度快、保密程度高是非对称密钥密码体制的优点之一
答案：错
解析：非对称密码运算量大、加解密速度慢；速度快是对称密码的特点，非对称密码优势是密钥管理、签名、密钥协商能力。
