# 信息泄露 - 备份文件下载 - 网站源码

# 题目介绍

这是一个教学题目，需要我们去找到并且下载网站的源码，但是需要注意的是网站源码的文件名和格式不统一，但是一般存在规律。

# 出题背景

当开发人员在线上环境中对源代码进行了备份操作，并且将备份文件放在了 web 目录下，就会引起网站源码泄露。

# 解题

进入环境后，发现出题人给了我们提示，对于新人还是非常友好的。
他告诉我们常见的网站源码备份文件后缀名有：tar、tar.gz、zip、rar，常见的网站源码备份文件名有：web、website、backup、back、www、wwwroot、temp。

于是思路就是，常见网站源码备份文件名和后缀名进行排列组合进行访问尝试
