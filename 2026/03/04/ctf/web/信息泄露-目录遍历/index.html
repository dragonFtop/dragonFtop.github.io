# 信息泄露 - 目录遍历

# 题目介绍

这是一个 Ctfhub 上面的一个 web 目录遍历的教学题目，环境是一个网站，我们只需要手动遍历访问网站的文件目录即可找到 flag

# 出题背景

有时候我们可以在网站的目录中遍历去寻找我们需要的东西。

# 解题

点击寻找 flag 后，进入题设的网站文件目录

在这个文件系统内遍历所有文件夹即可找到 flag.txt，其位置在 3/2 / 目录下
