# 替代密码

# 何为替代密码

  1. 替代密码是指先建立一个替换表，以查表得方式将需要加密的明文替换为密文，在解密时反向替换为明文。其中的替换表就是密钥。
  2. 根据使用的替换表数量，可以分为以下两种替换密码：
    （1）单表替换密码：加解密使用一个固定的替换表。
    （2）多表替代密码：加解密使用多个的替换表。
